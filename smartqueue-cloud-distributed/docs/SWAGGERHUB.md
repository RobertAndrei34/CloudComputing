# SwaggerHub Export Guide

The project already exposes the OpenAPI specification at:

```text
http://localhost:8000/openapi.json
```

## Steps

1. Start the project:

```bash
docker compose up --build
```

2. Open:

```text
http://localhost:8000/openapi.json
```

3. Save the page as `openapi.json`.

4. Go to SwaggerHub.

5. Create a new API.

6. Import the saved `openapi.json` file.

7. Publish or share the API documentation link.

8. Add the SwaggerHub link to this file.
